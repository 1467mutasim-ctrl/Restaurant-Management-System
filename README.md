# Restaurant-Management-System
A basic Restaurant Management System project written on Java. No database is required.


RESTAURANT MANAGEMENT SYSTEM

Run the project normally from NetBeans or other Java compiler (Run Project / F6).
The Java program starts a local web server and opens:

    http://localhost:8080

Credentials:
    Manager: manager / manager123
    Waiter:  waiter  / waiter123

No database is required. Menu items, reservations, orders and bills are kept
in memory and reset whenever the Java program is stopped.

If the browser does not open automatically, copy the address above into any
modern browser. Keep the Java program running while using the page. Stop it
from NetBeans when finished.
